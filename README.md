## Script Collection and Implementation

The scripts provided in this repository are designed to be executed via a browser's Developer Tools. Follow these steps to utilize them:

1. **Open the Course:** Navigate to your course in a modern web browser (Chrome, Edge, Firefox, or Safari).
2. **Open Developer Tools:** Press `F12` or `Ctrl+Shift+I` (`Cmd+Option+I` on macOS) once the page has loaded.
3. **Access the Console:** Select the **Console** tab within the Developer Tools window.
4. **Run the Script:** Paste the desired script into the console input field and press `Enter`.
5. **Observe Results:** Outcomes vary by script; please refer to any specific instructions or console output provided by the script.
